# dsipl1-claude-plugins-marketplace
Repo contains Claude plugins and a marketplace to serve them to Claude clients

## Usage

open your Claude Code CLI and run the following prompt:

*/plugin marketplace add https://github.com/biotronikdev/dsipl1-claude-plugins-marketplace.git*

once you added the marketplace, list the available plugins by typing

*/plugin*

install a plugin by typing

*/plugin install <plugin-name>@dsipl1-claude-plugins-marketplace*
